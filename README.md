# Expense Tracker

Expense Tracker is a responsive web application developed using **HTML, CSS, and JavaScript**. It helps users manage their personal finances by tracking income and expenses, monitoring the current balance, and viewing transaction history. The application stores data using **Local Storage**, allowing information to remain available even after the browser is closed.

## Features

* Add Income
* Add Expenses
* View Total Balance, Income, and Expenses
* Transaction History
* Delete Transactions
* Data Persistence with Local Storage
* Responsive Dashboard Design
* Clean and User-Friendly Interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## Purpose

This project was created to improve front-end development skills and demonstrate the ability to build a responsive, interactive, and real-world web application using core web technologies without any external frameworks.
